Header如下：

参数 值
Content-Type application/json; charset=utf-8

Body请求参数:

参数 类型 是否必须 说明
body array[{"key":String}] 是 搜索条件集合

[{"attribute": "id_0", "value": "1000", "flag": false}]

返回参数：

字段 类型 是否必须 说明
ids array[integer] 是 文件id编号

文件下载API


接口描述

根据搜索数据选择所需文件并下载数据。


接口使用说明

HTTP方法：POST

请求URL：http://localhost:8080/gjw/download

Header如下：

参数 值
Content-Type application/json; charset=utf-8

Body请求参数:

参数 类型 是否必须 说明
ids array[integer] 是 文件id编号

请求Body示例：

[0, 1]

返回说明

返回参数：

字段 类型 是否必须 说明
ids array[integer] 是 文件id编号

成功响应示例：弹出下载页面

失败响应示例：文件 :1.txt 不存在,请联系管理员!



数据上传API


接口描述

用户上传数据至临时表。


接口使用说明

HTTP方法：POST

请求URL：http://localhost:8080/gjw/add_products

Header如下：

参数 值
Content-Type application/json; charset=utf-8

Body请求参数:

参数 类型 是否必须 说明
数据名称 String 否 数据名称
DataType String 否
SizeX String 否
SizeY String 否
SizeZ String 否
... String 否

请求Body示例：

{ "id": 13, "name": "test1", "dataType": "", "sizeX": "1", "sizeY": "1", "sizeZ": "1", "strainX": "", "strainY": "1", "nx": "1", "ny": "", "elecX": "1", "elecY": "1", "elecZ": "", "xy_Fig": "undefined", "xz_Fig": "undefined", "xyz_Fig": "undefined", "data_File": "1.npz" }

返回说明

成功响应示例：页面刷新出现新增数据，数据库出现新增数据 status code=200

失败响应示例：上传失败
